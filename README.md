# User CRUD

A simple User CRUD with photo

## Requirements

- Java 11
- Maven
- Git 

## Installation

- `git clone https://github.com/rodrigot23/user_crud.git`
- `cd user_crud`
- `mvn clean install`

## Running

- `cd target`
- `java -jar user_crud-0.0.1.jar`.

## Usage

- http://localhost:8081/swagger-ui.html
